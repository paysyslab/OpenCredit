The Loan Origination System (LOS) interface is designed to provide a centralized and structured view of all activities related to loan processing.

It allows users to:

  • Monitor performance
  • Manage applications
  • Configure products
  • Track system activity

The interface is divided into key sections, each serving a specific operational purpose.

📊 Dashboard​

The Dashboard provides a high-level overview of system activity and performance.

It typically includes:

  • Application statistics
  • Approval/rejection trends
  • Operational insights
  • Key performance indicators (KPIs)

📂 Application Management​

This section is the core working area for operational teams.

It includes:

  • Loan Applications → View, review, and process loan requests
  • Onboarding Applications → Manage new customer registrations
  • Withdrawal Requests → Handle fund withdrawal-related actions

From here, users can:

  • Track application status
  • Review submitted details
  • Take actions based on workflow stages

🧩 Product Management​

This section controls how financial products are defined and behave within the system.

It includes configuration for:

  • Loan Product Setup
  • Fee Setup
  • KYC Groups
  • Interest / Markup Rules
  • Penalty Configuration
  • Tax Setup
  • Taxes on Fees
  • Credit Scoring Rules

These configurations determine:

  • How loans are structured
  • How charges are applied
  • How eligibility is assessed

🛡️ Audit Management​

Audit Management ensures transparency and traceability across the system.

It includes:

  • Loan Application Audits → Track changes and actions on applications
  • Backoffice Audits → Monitor user activity and system interactions

This helps in:

  • Ensuring compliance
  • Tracking user actions
  • Investigating discrepancies
